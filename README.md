# usage-data

Public, auto-refreshed data file powering [maxghenis.com/usage](https://maxghenis.com/usage).

Aggregated from local Claude Code and Codex JSONL session files via [Tokscale](https://tokscale.ai) and committed here every ~30 minutes.

Only contains daily aggregates: token counts, computed cost at list prices, and model breakdowns. No message content, no file paths, no project names.
